FingerROM AI v11

研究用試作版です。

主な変更点
- ページを開いている間だけ測定結果をメモリ保持します。
- 新しいWebタブを開くと、新しい患者（新しい症例コード）として開始します。
- 測定確定時にPNGやCSVを自動ダウンロードしません。
- 屈曲／伸展タブを追加しました。
- 左右・指・屈曲／伸展ごとの測定結果を、患者単位の1行にまとめてCSV出力します。
- 画面上のキャプチャ結果表示と手動修正方法はv10.1を維持しています。

注意
- ページを閉じる、再読み込みする、タブが破棄されると未出力データは消えます。
- 必要な測定後に「患者データをCSV出力」を押してください。
- 研究用プロトタイプであり、診断目的ではありません。
